How to — Creating new short URLs

Use the form above to submit a URL in the format: http(s)://www.example.com(/more/routes). The service will respond with a JSON object file containing your original long-form URL and a number for its short URL, for example: 

{"original_url": "https://www.website.com",
  "short_url": 1}

If you submit an invalidly formatted URL, or a website that doesn't exist, the microservice will respond with {"error": "invalid URL"} or {"error": "invalid hostname"}, respectively.

How to — Using the short URLs that you have generated

Navigate to https://pinto-potato.glitch.me/api/shorturl/<short_url>, replacing <short_url> with your desired short_url number. You will automatically be redirected to its matching long-form URL location.

If you type in an invalid short URL, the website will respond with a 404 error page. Note that all the short URL links generated by this service will be numbers.
